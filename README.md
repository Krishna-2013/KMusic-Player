## 🎵 KMusic Player

A Spotify-inspired music player web app built for learning frontend development.

⚠️ This project is not affiliated with or endorsed by Spotify. It is for educational purposes only.

---

## 🚀 Features
- ▶️ Play / Pause songs
- ⏭️ Next / Previous track
- 📊 Progress bar (seek functionality)
- 📁 Playlist UI

## 🎯 Purpose

KMusic Player is a personal project inspired by the design of Spotify.
It helps practice UI/UX, DOM manipulation, and interactive web development.
